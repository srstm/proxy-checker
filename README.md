# 🌐 Proxy Checker

A Python-based tool to validate and filter working proxies from a given proxy list. This project efficiently tests proxies for connectivity, speed, and usability, making it useful for web scraping, automation, and network testing tasks.

---

## 🚀 Overview

Free proxy lists often contain inactive or slow proxies. This project automates the process of:

- Checking proxy availability  
- Filtering working proxies  
- Measuring response performance  

It helps ensure only reliable proxies are used in applications like scraping, data collection, and automation pipelines.

---

## ✨ Features

- 🔍 Validate HTTP / HTTPS / SOCKS proxies  
- ⚡ Fast checking using multi-threading  
- 🧪 Test proxy connectivity with real requests  
- 📊 Filter active vs inactive proxies  
- 📁 Save working proxies to output files  
- ⚙️ Easy to customize and extend  

---

## 🏗 Tech Stack

- **Language:** Python  
- **Libraries Used:**
  - Requests  
  - Threading  
  - Socket (optional for low-level checks)
